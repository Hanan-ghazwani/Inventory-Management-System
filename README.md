# نظام إدارة المخزون (متعدد الصلاحيات) | Inventory Management System

نظام إداري متكامل لإدارة المخزون والطلبات داخل المنشأه ، تم تطويره برمجياً لضمان سلاسة سير العمل وتوزيع المهام بدقة.

##  المميزات الرئيسية | Key Features
* تعدد الأدوار (RBAC): نظام صلاحيات مخصص لـ (المسؤول، الموظف، الفني، موزع الطلبات).
* إدارة الطلبات: إمكانية إسناد المهام للفنيين وتتبع حالة الطلب (قيد التنفيذ، مكتمل، ملغي).
* نظام القيود: تحديد حد أقصى (3 طلبات يومياً) لكل موظف لضمان كفاءة العمل.
* لوحة تحكم تفاعلية: تحديثات فورية للمخزون وتوزيع الطلبات و تتبع الطلبات.
